# South Forsyth Evergreen Content Strategy

This document outlines an evergreen content strategy for SouthForsyth.org focused on repeatedly searched local topics rather than one-off blog posts. The goal is to build topical authority around practical, high-intent searches that can support long-term organic growth.

## Strategy principle
Create large, useful guides that answer the same questions people search for again and again. These pages should become reference points that earn links, repeat visits, and strong visibility in search and AI-driven discovery.

### Why these pages come first
- They match high-intent local search behavior.
- They are evergreen and can be updated year-round.
- They naturally support internal linking across the site.
- They create strong topical clusters for discovery and AI search.

## Suggested content model
Each evergreen guide should include:
- Hero
- Introduction
- Table of contents
- Suggested sections
- Internal links
- FAQ
- Related guides
- Newsletter CTA
- Schema TODO comments
- Dynamic content TODO comments

## Internal linking plan
- Link each evergreen guide back to the main hub pages.
- Link related guides together in a natural way.
- Use sidebar blocks and footer links to connect the content clusters.
- Make each page a gateway into the next relevant resource.

## SEO and AI strategy
- Focus each page on one clear primary search intent.
- Use structured, answer-oriented sections with helpful FAQs.
- Keep content organized for both search engines and AI summarization.
- Update pages regularly with current local details.

## Recommended rollout order
1. Launch the highest-intent local discovery pages first.
2. Publish supporting guide pages that link to them.
3. Expand into seasonal and event-based guides once the core evergreen structure is established.
4. Add local business and directory data to deepen the content network.
